Welcome to the official Relic documentation. For organizational purposes, this
document is divided up into 4 sections:
+ Tutorial :: The tutorial (incomplete) is a basic walk-through of the language
  for programmers new to Relic. 
+ Cookbook :: The Cookbook section contains a set of 'recipes', common tasks
  such as reading a file or printing a table together with example implementations.   
+ Discussions :: The discussions section includes deep-dives into the reasoning
  (the 'whys') behind various aspects of the language.
+ Spec :: The specification allows tree-like navigation of the entire standard
  library, with basic descriptions of each value, type and term-former. It is
  the most complete, and is most suted to those familiar with the language.
