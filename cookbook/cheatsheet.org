* Literals
Integers: 0, 1, -1, 64, etc.
+ default to type I64
+ use =is= or inference to change this default

Strings: "hello,world!"
+ currently no special characters (\n etc.)
+ live for as long as their parent expression or definition

* Structs  
#+begin_src relic
;; Struct type (structural/anonymous)
(def Point Struct
  [.x I64]
  [.y I64] [.z I64])

(def point struct [.x 3] [.y 4] [.z 5])

(i64.+ point.x point.y)
#+end_src

* Dynamic
#+begin_src relic
(def dvar dynamic 10) ;; new dynamic variable

(use dvar) ;; => 10

(def get-dvar proc [] (use dvar))

(bind [x 3] (get-dvar)) ;; => 3

(use dvar) ;; => 10
#+end_src
