* Macros
+ list :: A basic macro for creating literal lists. Use as ~(list elt1 elt2 elt3)~.
  For example, ~(list 1 2 3 4 5)~ will generate a list containing the elements
  1-5. The list macro uses the =mk-list= function, and thus allocates memory
  with =current-allocator=.
