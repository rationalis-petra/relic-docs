+ create-buffer ::
#+begin_src relic
Proc [InputRate U64] Buffer
#+end_src
  Creates a buffer on the device, 
  - buffer-type : indicate HOW data is to be fed into shaders, with
    vertex indicating that it stores vertices, index indicating that it is
    indexing /into/ the vertex buffer, Uniform indicates that this is global
    for a single invocation.
