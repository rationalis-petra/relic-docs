Syntactic Terms/Formers

def
Define a new value. Can only be used as a toplevel term in a module.
declare
Declare that a value has certain properties. Canonly be used as a toplevel term in a module.
import
Can only be used in scripts.
proc
Create a new procedure (function).
all
Create a new universally quantified behaviour.
apply
explicitly apply a function to some value(s).
seal
Create a new term that will seal away a type from a value.
unseal
macro
Convert a procedure into a macro.
